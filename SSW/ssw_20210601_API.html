HTTP 란?¶

  • Hyper Text Transfer Protocol
  • 인터넷에서 데이터를 주고받을 수 있는 프로토콜(규칙)
  • 요청(Request, 입력)와 응답(Response, 출력)로 구성
    • 요청: Client --> Server
    • 응답: Server --> Client

응답 데이터 형식¶

JSON(JavaScript Object Notation)

  • Key-Value 의 쌍으로 이루어진 구조
  • 파이썬의 딕셔너리 자료형과 유사함

XML(Extensible Markup Language)

  • HTML 과 유사함

https://ko.wikipedia.org/wiki/XML

Python 으로 JSON 데이터 다루기¶

JSON to Dictionary¶

loads() 함수: JSON 문자열을 Python 딕셔너리 객체로

In [ ]:
import json

json_string = '''{
    "name": "홍길동",
    "gender": "남성",
    "age": 50
}'''

json_object = json.loads(json_string)

print(json_object)
print(type(json_object))

print(json_object.get('name'))
print(json_object.get('gender'))
print(json_object.get('age'))

Dictionary to JSON¶

dumps() 함수: Python 딕셔너리 객체를 JSON 문자열로

In [ ]:
import json

json_object = {
    "name": "홍길동",
    "gender": "남성",
    "age": 50
}

json_string = json.dumps(json_object, indent=4, ensure_ascii=False)

print(json_string)
print(type(json_string))
